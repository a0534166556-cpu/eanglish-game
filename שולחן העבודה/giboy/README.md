# English Learning Game

אפליקציית משחק ללימוד אנגלית עם מגוון משחקים ואתגרים.

## תכונות עיקריות

- **מערכת התחברות** - הרשמה והתחברות למשתמשים
- **בחירת רמה** - התאמת התוכן לרמת המשתמש
- **מגוון משחקים**:
  - אמת מילים (Matching Pairs)
  - השלמת משפטים
  - הזזת מילים
  - בחירה מרובה
  - נכון/לא נכון
  - האזנה וקליטה
  - הגייה
  - תיאור תמונה
- **חנות** - קניית פריטים מיוחדים בנקודות
- **מערכת נקודות** - מעקב אחר התקדמות

## טכנולוגיות

- Next.js
- React
- TypeScript
- Tailwind CSS
- Firebase
- Zustand
- Framer Motion

## התקנה

1. התקן את התלויות:
```bash
npm install
```

2. הפעל את שרת הפיתוח:
```bash
npm run dev
```

3. פתח את הדפדפן בכתובת:
```
http://localhost:3000
```

## מבנה הפרויקט

```
/app
  /components
    /auth - קומפוננטות התחברות
    /games - קומפוננטות משחקים
    /shop - קומפוננטות חנות
    /common - קומפוננטות משותפות
  /pages
    /login - דף התחברות
    /level-select - בחירת רמה
    /games - דף משחקים
    /shop - חנות
  /styles
  /utils
  /types
```
