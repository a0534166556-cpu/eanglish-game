# 🏆 מערכת הישגים מקיפה

## סקירה כללית

מערכת הישגים מתקדמת עם 57+ הישגים מכל רמות הקושי, שמשפיעים ישירות על רמת השחקן בפרופיל.

## 📊 רמות קושי

### 🟢 קל (Easy) - 10 הישגים
- **פרסים:** 10-75 💎
- **נקודות ניסיון:** 50-250 XP
- **דוגמאות:**
  - צעדים ראשונים (משחק ראשון)
  - ניצחון ראשון
  - שבוע שלם (7 ימים רצופים)
  - רמה 5

### 🟡 בינוני (Medium) - 14 הישגים
- **פרסים:** 100-300 💎
- **נקודות ניסיון:** 500-2000 XP
- **דוגמאות:**
  - שחקן מנוסה (25 משחקים)
  - מאסטר המשחקים (50 ניצחונות)
  - חודש שלם (30 ימים רצופים)
  - רמה 20
  - אוהב אנגלית (100 שאלות נכונות)

### 🟠 קשה (Hard) - 14 הישגים
- **פרסים:** 500-1000 💎
- **נקודות ניסיון:** 5000-10000 XP
- **דוגמאות:**
  - שחקן ותיק (200 משחקים)
  - אגדה חיה (200 ניצחונות)
  - שלושה חודשים (90 ימים רצופים)
  - רמה 50
  - מומחה אנגלית (500 שאלות נכונות)
  - מאסטר Word Clash (50 ניצחונות)

### 🔴 אקסטרים (Extreme) - 19 הישגים
- **פרסים:** 2000-10000 💎
- **נקודות ניסיון:** 20000-100000 XP
- **דוגמאות:**
  - מיתוס המשחקים (1000 משחקים)
  - אלוהי הניצחון (1000 ניצחונות)
  - שנה שלמה (365 ימים רצופים)
  - נצחי (500 ימים רצופים)
  - רמה 100 (רמה מקסימלית!)
  - פרופסור אנגלית (2000 שאלות נכונות)
  - מיליונר מטבעות (מיליון מטבעות)
  - מלך היהלומים (10,000 יהלומים)
  - אגדת הלמידה (השלמת כל הרמות)

## 🎯 קטגוריות

### 🎮 משחקים (Games)
הישגים הקשורים למספר משחקים ששוחקו וניצחונות.

### 🔥 רצף (Streak)
הישגים הקשורים למשחק רצוף במשך ימים.

### ⭐ רמה (Level)
הישגים הקשורים להגעה לרמות שונות.

### 💎 מיוחד (Special)
הישגים ייחודיים כמו:
- מספר שאלות נכונות
- למידת מילים חדשות
- קניות בחנות
- הישגים ספציפיים למשחקים מסוימים

## 💰 מערכת פרסים

### יהלומים (💎)
- מתקבלים מיידית עם השלמת הישג
- ניתן להשתמש בהם בחנות

### נקודות ניסיון (⭐ XP)
- **מתווספות ישירות לרמת השחקן**
- משפיעות על הדרגה בפרופיל
- ככל שההישג קשה יותר, כך יותר XP

## 🎨 תצוגה

כל הישג מציג:
- 🎯 **אייקון** - ייצוג ויזואלי של ההישג
- 🏷️ **קטגוריה** - סוג ההישג
- 🎨 **רמת קושי** - עם צבע מתאים
- 📊 **התקדמות** - פס התקדמות ויזואלי
- 💎 **פרס יהלומים** - כמות היהלומים שמתקבלים
- ⭐ **פרס XP** - כמות נקודות הניסיון
- ✅ **סטטוס** - האם הושלם או לא
- 📅 **תאריך השלמה** - מתי הושלם ההישג

## 🔄 עדכון אוטומטי

ההישגים מתעדכנים אוטומטית:
1. **בסיום משחק** - עדכון התקדמות
2. **בניצחון** - עדכון מונה ניצחונות
3. **בהתחברות יומית** - עדכון רצף
4. **ברכישה בחנות** - עדכון מונה רכישות
5. **בלמידת מילה חדשה** - עדכון אוצר מילים

## 📈 השפעה על רמת השחקן

### חישוב רמה כולל:
```javascript
totalScore = basePoints + winBonus + playBonus + winRateBonus + achievementsXP
```

- **basePoints** - נקודות בסיס מהמשחקים
- **winBonus** - בונוס על ניצחונות (50 נקודות לניצחון)
- **playBonus** - בונוס על משחקים (10 נקודות למשחק)
- **winRateBonus** - בונוס על אחוז ניצחונות
- **achievementsXP** - **נקודות מהישגים שהושלמו!**

### דרגות:
1. 🌱 **מתחיל** - 0 נקודות
2. 📚 **לומד** - 500 נקודות
3. 🎓 **תלמיד** - 1,500 נקודות
4. ⚡ **מיומן** - 3,500 נקודות
5. 🌟 **מומחה** - 7,000 נקודות
6. 💎 **אמן** - 12,000 נקודות
7. 👑 **אמן גדול** - 20,000 נקודות
8. 🏆 **אלוף** - 35,000 נקודות
9. ⭐ **אגדה** - 60,000 נקודות
10. 🌠 **מיתוס** - 100,000 נקודות

## 🚀 איך להשיג הישגים

### טיפים:
1. **שחק באופן קבוע** - רצף יומי מזכה בהישגים רבים
2. **נצח במשחקים** - הישגי ניצחונות מעניקים הרבה XP
3. **נסה משחקים שונים** - הישגים ספציפיים למשחקים
4. **למד מילים חדשות** - הישגי אוצר מילים
5. **קנה בחנות** - הישגי אספנות
6. **שאף לרמות גבוהות** - הישגי רמה מעניקים הרבה פרסים

### אסטרטגיה:
- התחל עם הישגים קלים לצבור מומנטום
- עבוד על הישגי רצף - הם מצטברים עם הזמן
- התמקד בהישגים בינוניים לצבור XP יציב
- שאף להישגים קשים ואקסטרים לפרסים גדולים

## 🛠️ טכני

### מבנה הדאטאבייס:
```prisma
model Achievement {
  id          String
  name        String
  description String
  icon        String
  category    String
  requirement Int
  reward      Int
  difficulty  String  // easy, medium, hard, extreme
  xpReward    Int     // נקודות ניסיון
  isActive    Boolean
  createdAt   DateTime
}

model UserAchievement {
  id            String
  userId        String
  achievementId String
  progress      Int
  isCompleted   Boolean
  completedAt   DateTime?
}
```

### API Endpoints:
- **GET** `/api/achievements?userId={id}` - קבלת כל ההישגים עם התקדמות
- **POST** `/api/achievements` - עדכון התקדמות והשלמת הישגים

## 📝 הוספת הישגים חדשים

להוספת הישגים חדשים, ערוך את `scripts/add-achievements.js` והרץ:
```bash
node scripts/add-achievements.js
```
